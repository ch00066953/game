# 扫雷

一个原生 HTML/CSS/JavaScript 的扫雷小游戏。

## 已实现功能

- 默认初级难度：9x9，10 雷
- 难度切换：初级 / 中级 / 高级
- 自定义行列雷数开局
- 首次点击保护（首点必非雷）
- 左键翻开，右键循环标记（🚩 -> ❓ -> 未标记）
- 双击数字格快速翻开（满足插旗数等于数字）
- 胜负判定与反馈
- 计时器（首次翻开后开始）
- 剩余雷数显示（总雷数 - 插旗数）
- 命令输入支持：
  - `/mine`、`/mine 初级`、`/mine 中级`、`/mine 高级`
  - `开始扫雷`、`重来`、`新一局`
  - `点开 行 列`（如：点开 3 5）

## 运行方式

直接在浏览器打开 `index.html` 即可。
