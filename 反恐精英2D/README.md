# 反恐精英 2D

俯视角双人合作射击游戏，纯原生 HTML / CSS / JavaScript + Canvas 实现。

## 游戏模式

- **单人模式** — 独自对抗 AI 敌人，共 5 波
- **双人合作** — 两名玩家同屏键盘操作，组队消灭 AI 敌人

## 操作说明

| 功能 | P1 | P2 |
|------|-----|-----|
| 移动 | W A S D | ↑ ↓ ← → |
| 射击 | Space | / |
| 换弹 | R | . |
| 切换武器 | E | , |
| 暂停 | ESC | ESC |

## 地图

- **仓库** — 对称布局，散落箱体掩体
- **巷战** — 狭窄通道与交叉路口
- **广场** — 开阔中央区域，四周建筑环绕

## 武器

| 武器 | 伤害 | 弹匣 | 特性 |
|------|------|------|------|
| 手枪 | 25 | 12 | 无限备弹，默认武器 |
| 步枪 | 18 | 30 | 高射速，拾取获得 |
| 霰弹枪 | 14×5 | 8 | 散射，近距离强力 |

## 敌人类型

- **士兵** — 普通敌人，手枪攻击
- **重装兵** — 高血量，霰弹枪
- **狙击手** — 远程高伤害，血量低

## 技术栈

纯前端：HTML5 Canvas + 原生 JavaScript，零依赖。
