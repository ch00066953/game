# Plants vs Zombies 2 - Mini

A lightweight browser mini game inspired by lane-based plant defense gameplay.

## New Features

- 3 plant cards:
	- Pea (100): basic single shot
	- Wallnut (75): high HP blocker
	- Repeater (175): double shot
- 3 zombie types:
	- normal
	- cone
	- bucket
- Falling suns:
	- click suns on the board to collect +25 sun
- Simple sound effects with On/Off toggle

## How to Play

1. Open `index.html` in a browser.
2. Select a plant card in the control bar.
3. Click a grid cell to place the plant.
4. Click falling suns to gain more resources.
5. Stop zombies from reaching the left edge.
6. Defeat 24 zombies to win.

## Files

- `index.html`: page layout and HUD
- `styles.css`: visual style and responsive layout
- `game.js`: gameplay loop, render, and interactions
