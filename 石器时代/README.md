# 石器时代

一款纯前端实现的回合制 RPG 游戏，致敬经典网游《石器时代》。

## 游戏特色

- **3 大职业**：战士、猎人、巫师，各有独特技能树
- **宠物系统**：捕获野生恐龙作为战斗伙伴，最多携带 6 只
- **回合制战斗**：攻击、技能、捕获、道具、换宠、逃跑 六大指令
- **5 大区域**：加加村 → 密林深处 → 灼热荒漠 → 烈焰火山 → 极寒冰原
- **15 种怪物**：从小恐龙到冰龙王，每种都可捕获为宠物
- **等级成长**：角色 & 宠物独立升级，学习新技能
- **部落商店**：药水、捕兽石、增益道具
- **本地存档**：随时保存进度

## 操作说明

1. 创建角色，选择职业
2. 点击「探索」进行野外遭遇战
3. 战斗中选择指令：攻击消灭敌人或捕获为宠物
4. 升级后前往新区域挑战更强敌人
5. 在商店购买道具辅助冒险

## 技术实现

- 纯原生 HTML + CSS + JavaScript
- Canvas 绘制场景地图
- LocalStorage 本地存档
- CSS 动画实现战斗特效
