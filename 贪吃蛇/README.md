# 🐍 贪吃蛇游戏

一个功能完整的网页版贪吃蛇游戏，使用 HTML、CSS 和原生 JavaScript 开发。

## 功能特性

✨ **核心游戏功能**
- 🎮 用方向键或 WASD 控制蛇的方向
- 🎯 吃掉红色食物增加分数和身体长度
- 📊 实时显示当前分数、最高分和游戏难度
- 🚀 每吃掉 10 个食物，难度自动增加（速度加快）

🎛️ **游戏控制**
- **开始游戏** - 开始新游戏
- **暂停** - 暂停和继续游戏（可按空格键）
- **重新开始** - 重置游戏状态

💾 **数据持久化**
- 自动保存最高分到本地存储
- 刷新页面后最高分不会丢失

🎨 **用户界面**
- 美观的梯度背景和现代化设计
- 响应式布局，支持移动设备
- 清晰的游戏信息显示
- 实时反馈消息

## 游戏规则

1. 蛇初始位置在游戏板中心
2. 使用以下按键控制蛇的方向：
   - **↑ 向上** / **W**
   - **↓ 向下** / **S**
   - **← 向左** / **A**
   - **→ 向右** / **D**
   - **空格** - 暂停/继续

3. **目标** - 吃掉红色食物（每个 +10 分）
4. **游戏结束** - 当蛇撞到墙壁或自己的身体时
5. **难度提升** - 每吃掉 10 个食物，游戏速度增加

## 如何运行

### 方式 1：直接打开 HTML 文件
1. 在文件管理器中找到 `index.html`
2. 双击打开或用浏览器打开

### 方式 2：使用 VS Code 扩展
1. 安装 "Live Server" 扩展
2. 右键点击 `index.html`，选择 "Open with Live Server"

### 方式 3：本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js (需要安装 http-server)
npx http-server

# 然后在浏览器中访问
# http://localhost:8000
```

## 技术细节

### 游戏状态管理
- `gameState` 对象跟踪所有游戏状态
- `snake` 数组存储蛇的所有身体段
- `direction` 和 `nextDirection` 处理输入缓冲

### 核心算法
- **网格系统** - 20×20 的网格
- **碰撞检测** - 墙壁和自身碰撞检查
- **食物生成** - 随机生成不与蛇重叠的食物
- **难度系统** - 根据得分动态调整游戏速度

### 浏览器特性使用
- `localStorage` - 存储最高分
- `setInterval` - 游戏循环
- `addEventListener` - 键盘输入处理
- CSS Grid - 游戏板布局
