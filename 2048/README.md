# 2048

经典数字滑块游戏，合并相同数字挑战 2048。

## 玩法

- 滑动使所有方块向一个方向移动
- 相同数字的方块碰撞会合并为它们的总和
- 每次滑动后随机出现一个新方块（90% 为 2，10% 为 4）
- 合出 2048 即为胜利，可继续挑战更高分
- 无法移动时游戏结束

## 操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动 | ↑ ↓ ← → / WASD | 触摸滑动 |
| 新游戏 | 按钮 | 按钮 |

## 特性

- 本地最高分持久化
- 触控滑动检测
- 方块弹出与合并动画
- 响应式布局，自适应屏幕
- 经典配色方案

## 已知限制

- 无撤销功能
- 无音效
