# 👻 吃豆人（Pac-Man）

一个使用原生 HTML/CSS/JavaScript 实现的网页版吃豆人。

## 已实现能力

- 经典迷宫网格（21x21）、墙体/通道/豆子/能量豆
- 吃豆人四方向移动 + 预输入转向系统
- 4 个幽灵（Blinky / Pinky / Inky / Clyde）配备独立 AI
- 幽灵模式：`chase` / `scatter` / `frightened`（含闪烁警告）
- 自动墙体规范化：消除无法到达的死角，确保全部豆子可收集
- 得分、关卡、生命系统
- 8 个关卡递进难度（幽灵更快、frightened时间缩短）
- 通关后进入下一关并提升速度，最终完成第8关胜利
- 果实奖励（按吃豆进度生成）
- 开始菜单、暂停、结束面板、音效与背景音系统

## 新增高级特性（v2.0+）

### 🎮 关卡动态性体验
- **关卡过渡提示**：每关通过后暂停 2.2 秒并显示下一关提示，避免突兀跳关
- **关卡地图变异**：每一关随机打开 1-2 道墙壁，增加每次游玩的独特性
- **动态传送门**：两个传送门（紫色/青色）每 8-12 秒中随机变换一个位置，增加不可预测性

### 💣 随机炸弹系统
- **自动定时生成**：每 8-13 秒随机在可行走的地方生成一枚倒计时炸弹
- **爆炸计时逻辑**：炸弹生成后 5 秒爆炸，最后 1.5 秒闪烁警告提示危险
- **爆炸影响范围**：3×3 区域（中心周边 1.5 格），对角色和幽灵都有伤害
- **伤害效应**：
  - 🚫 **Pac-Man 接触爆炸** → 扣一条生命，启动死亡动画
  - 👻 **幽灵接触爆炸** → 非 frightened 模式下返回巢穴；frightened 模式下被吃掉
- **视觉与音效**：
  - 倒计时显示火焰特效
  - 爆炸时播放低沉爆炸音效
  - 被击飞幽灵显示浮动文字反馈

### 🎁 随机能量物品系统
4 种类型物品，每 7-20 秒自动生成一个，最多存活 10 秒：

1. **幸运奖励** 🟡（黄色钻石）- 瞬间获得 **+180 分**，即时反馈
2. **幽灵减速** 🔵（青色钻石）- 所有幽灵速度降至 **0.78 倍**，持续 **6 秒**
3. **双倍得分** 🟠（橙色钻石）- 豆子分值翻倍（10→20，50→100），持续 **8 秒**
4. **迷雾效应** 🟣（紫色钻石）- 屏幕覆盖半透明迷雾，视线受阻，持续 **6 秒**

所有物品收集时伴随浮动提示文字与音效反馈。

## 运行方式

直接打开 `index.html`，或使用本地静态服务器（如 Live Server）。

## 操作说明

- **方向键** ↑↓←→：移动 / 预输入转向
- **空格** Space：暂停 / 继续游戏
- **"如何玩"按钮**：查看控制提示与游玩指南

## 游戏难度进阶

随着关卡升高：
- 🚴 **1-3 关**：基础难度，幽灵速度 4.6-4.9 单位/秒，frightened 时间 ~5-6 秒
- 🏃 **4-6 关**：中等难度，幽灵速度 5.1-5.5 单位/秒，frightened 时间 ~3-4 秒
- 🚀 **7-8 关**：极限难度，幽灵速度 6.0-6.3 单位/秒，frightened 时间 ~3.5 秒以下

额外挑战：炸弹频率逐关稳定，传送门随时位移，能量物品出现频率保持恒定。

## 技术亮点

- **纯原生 JavaScript**：无依赖库，清晰的函数式架构
- **高效渲染**：Canvas 2D，60 FPS 目标，平滑动画
- **图论算法**：6-阶段 BFS 图规范化，自动消除陷阱区域
- **AI 寻路**：距离启发式 + 模式切换，幽灵行为拟真
- **状态机设计**：模式调度、倒计时管理、碰撞分层检测
