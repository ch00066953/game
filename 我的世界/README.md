# 我的世界：像素沙盒

一个使用原生 HTML、CSS、JavaScript 编写的简化版“我的世界”网页小游戏。

## 功能特性

- 程序生成方块地形，包含平原、森林、高地、沙岸和水域
- 等距像素视角渲染，支持昼夜循环与小地图
- 使用 WASD 或方向键移动角色
- 左键采集、右键放置，支持 6 种可堆叠方块
- 本地自动存档，也支持手动保存和清空进度

## 操作说明

- 移动：WASD 或方向键
- 挖掘：鼠标左键
- 放置：鼠标右键
- 切换快捷栏：数字键 1-6
- 重新生成世界：R
- 保存进度：F
