# ⛏️ 黄金矿工

经典黄金矿工游戏的网页复刻版，使用原生 HTML / CSS / JavaScript（Canvas）实现。

## 玩法

1. 钩子会左右自动摆动
2. 按 **↓ / 空格 / 点击屏幕** 释放钩子
3. 钩子碰到矿物后自动抓取并回收
4. 在规定时间内达到目标金额即可过关
5. 回收中按 **↑ / 再次点击** 可使用炸药丢弃石头

## 矿物类型

| 矿物 | 价值 | 重量 | 说明 |
|------|------|------|------|
| 小金块 | $50 | 轻 | 最常见 |
| 中金块 | $100 | 中 | 价值平衡 |
| 大金块 | $250 | 重 | 高价值但回收慢 |
| 💎 钻石 | $600 | 极轻 | 稀有高价值 |
| 石头 | $10 | 重 | 不值钱又沉 |
| 大石头 | $20 | 极重 | 尽量避免 |
| 👜 神秘袋 | 随机 | 轻 | 开出 $50~$300 |

## 商店道具

- 💪 **力量药水**（$30）：下一次回收速度提升 80%
- 🧨 **炸药**（$20）：回收石头时可按 ↑ 炸掉

## 技术实现

- Canvas 2D 绘图
- 钩子物理摆动 + 伸缩动画
- 矿物碰撞检测
- 粒子特效系统
- 关卡递进难度
- 移动端触摸支持
