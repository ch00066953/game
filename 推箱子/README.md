# 推箱子（Sokoban）

经典推箱子益智游戏，将所有箱子推到目标位置即可过关。

## 玩法

- 控制角色在地图上移动
- 推动箱子到标有 ◇ 的目标位置上
- 箱子只能推不能拉，且不能推动两个箱子
- 所有箱子都到达目标位置即过关
- 内置 10 个关卡，难度递增

## 操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动 | ↑ ↓ ← → / WASD | 虚拟方向键 / 滑动 |
| 撤销 | Z / Ctrl+Z | 撤销按钮 |
| 重置 | R | 重置按钮 |
| 切换关卡 | 上/下一关按钮 | 上/下一关按钮 |

## 特性

- 10 个经典关卡
- 撤销功能（无限步数）
- 关卡进度本地保存
- 虚拟方向键 + 滑动操控
- 响应式布局

## 已知限制

- 无音效
- 无自定义关卡编辑器
