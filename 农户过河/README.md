# 👨‍🌾 农户过河（狼、羊、菜）

基于经典“狼、羊、菜过河谜题”的网页小游戏，使用 HTML + CSS + 原生 JavaScript 实现。

## 玩法规则

1. 农夫要把狼、羊、白菜从左岸送到右岸。
2. 每次农夫过河只能带 0 件或 1 件物品。
3. 农夫不在场时：
- 狼和羊同岸会发生“狼吃羊”。
- 羊和白菜同岸会发生“羊吃白菜”。
4. 目标：让四者全部安全到达右岸。

## 已实现功能

- 命令交互：`/river`、`开始过河谜题`、`带狼`、`带羊`、`带菜`、`独自`、`状态`、`提示`、`解法`、`重置`
- 快捷按钮与命令行输入两种操作方式
- 安全校验（失败原因提示）
- 胜利判定与步数统计
- 基于 BFS 的“提示”与“最优解”展示

1. 直接用浏览器打开 `index.html`。
2. 或使用 Live Server 启动。

## 最优解（7步）

1. 带羊到右岸
2. 独自回左岸
3. 带狼到右岸
4. 带羊回左岸
5. 带菜到右岸
6. 独自回左岸
7. 带羊到右岸
