# 魂斗罗（完整强化版）

原生 HTML/CSS/JavaScript 实现的横版卷轴射击小游戏，包含精细角色绘制、武器系统与 Boss 战。

## 核心特性

### 🎨 视觉呈现
- **分层动态背景**：月光、远山、城市剪影、视差滚动
- **主角精细绘制**：头巾、护肩、武器、跑动摆动动画
- **小怪精细绘制**：机甲头盔、胸甲渐变、面罩灯光
- **肌肉男 Boss**：高血量、呼吸动画、肌肉高光细节
- **粒子特效系统**：开火火花、命中特效、击杀爆裂
- **屏幕震动反馈**：受伤与 Boss 砸地冲击感

### 🔫 武器系统
- **普通枪**：基础武器，10帧冷却
- **散弹枪** (S)：三发散射，覆盖面积大
- **激光枪** (L)：高伤害穿透，5帧冷却
- **追踪弹** (H)：自动锁定敌人，高伤害

### 💎 道具掉落
- **生命包** ❤️：恢复2点生命（小怪35%掉落率）
- **武器道具**：散弹/激光/追踪，持续10秒

### 👹 Boss 战斗
- **入场演出**：140帧倒计时+屏幕震动
- **追击系统**：跟踪玩家位置
- **三段攻击**：
  - 重火力射击（狂暴时3连发）
  - 跳跃砸地（产生冲击波）
  - 狂暴模式（血量<50%触发）
- **Boss 血条**：世界坐标显示+HUD实时显示

### 🏆 通关奖励
- **胜利场景**：Canvas 绘制金发美女与主角亲吻场景
- **动画效果**：金色光晕、爱心飘落、角色倾斜动画
- **5秒演出**：渐入黑屏+角色绘制+特效叠加

## 操作说明

- `A/D` 或 `←/→`：移动
- `W` / `↑` / `Space`：跳跃
- `J`：射击
- `P`：暂停/继续

## 胜利条件

1. 清空所有小怪
2. 击败肌肉男 Boss
3. 抵达终点线

## 运行方式

直接打开 `index.html`，或使用 Live Server 运行。
