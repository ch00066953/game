# 打砖块（Breakout）

经典打砖块游戏，操控挡板反弹小球击碎所有砖块。

## 玩法

- 移动挡板接住小球，让小球反弹击碎上方砖块
- 击碎所有砖块即可进入下一关
- 小球掉落底部会损失一条命，3 条命用完游戏结束
- 随关卡提升，球速增加，砖块血量增加

## 操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动挡板 | ← → / A D | 触摸滑动 Canvas 或底部按钮 |
| 发射小球 | 空格 / 点击 | 点触 Canvas 或发射按钮 |
| 暂停 | 空格 | — |
| 重玩 | 重玩按钮 | 重玩按钮 |

## 特性

- Canvas 渲染，流畅动画
- 多关卡递进难度
- 砖块血量系统
- 挡板击球角度控制
- 键盘与触控双模式
- 响应式布局，移动端友好

## 已知限制

- 无音效
- 无本地存档
