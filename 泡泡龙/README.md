# 泡泡龙（Puzzle Bobble）

经典泡泡龙射击消除游戏，发射泡泡使三个以上同色相连即可消除。

## 玩法

- 从底部发射泡泡到顶部泡泡阵列
- 3 个及以上相连的同色泡泡会被消除
- 消除后失去连接的浮空泡泡也会掉落
- 泡泡堆积到底部则游戏结束
- 清除所有泡泡进入下一关

## 操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 瞄准 | 鼠标移动 / ← → | 触摸拖动 |
| 发射 | 点击 / 空格 / ↑ | 松手 |
| 重玩 | 重玩按钮 | 重玩按钮 |

## 特性

- 六角网格泡泡排列
- 瞄准线指引
- 浮空泡泡自动掉落
- 多关卡递进难度（颜色增多）
- 鼠标、键盘、触控三模式

## 已知限制

- 无音效
- 固定关卡布局（随机生成）
