## 推荐实现优先级（小团队/独立开发）
1. **最快可做原型**：打砖块、吃豆人、2048、推箱子、扫雷  
2. **中等复杂度**：塔防基础版、平台跳跃、回合战棋小地图  
3. **高复杂度**：RPG（含剧情/存档）、联网多人、开放世界沙盒

## 你可以下一步这样定
- 先选 **1 个核心玩法**（例如“塔防”）
- 再定 **最小可玩版本（MVP）**：1张地图 + 3种敌人 + 2种防御塔
- 最后补 **可迭代项**：商店、技能树、关卡编辑器、排行榜
