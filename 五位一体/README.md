# 五位一体 - 3D冒险游戏

## 游戏概述
这是一款基于物理引擎的3D冒险游戏，灵感来源于《三位一体》游戏，但扩展为五个可切换的角色：
- 战士：近战攻击，高防御
- 法师：魔法攻击，解谜能力
- 盗贼：敏捷移动，解锁技能
- 工匠：建造和修复物品
- 商人：交易和资源管理

## 技术栈
- Three.js: 3D渲染引擎
- Cannon.js: 物理引擎
- 原生JavaScript: 游戏逻辑

## 功能特性
- 5个独特角色，每个角色有不同能力和玩法
- 真实的物理交互系统
- 角色切换机制
- 解谜元素
- 交易和制作系统

## 安装运行
直接打开index.html文件即可运行游戏（需要WebGL支持）
