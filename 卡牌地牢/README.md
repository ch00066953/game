# 卡牌地牢（Card Roguelike）

受"杀戮尖塔"启发的卡牌 Roguelike 游戏，构建牌组击败地牢中的敌人。

## 玩法

- 每回合获得 3 点能量，抽 5 张手牌
- 打出攻击牌对敌人造成伤害
- 打出防御牌获得护盾（回合开始时清零）
- 使用技能牌获得增益或回复
- 击败敌人后从 3 张奖励中选择加入牌组
- 共 8 层地牢，逐层增强的敌人

## 卡牌

| 卡牌 | 费用 | 效果 |
|------|------|------|
| 打击 | 1 | 6 伤害 |
| 猛击 | 2 | 12 伤害 |
| 横扫 | 1 | 全体 4 伤害 |
| 防御 | 1 | 5 护盾 |
| 铁壁 | 2 | 12 护盾 |
| 治疗 | 1 | 回复 6 HP |
| 淬毒 | 1 | 3 伤害 + 3 毒 |
| 强化 | 0 | +2 力量 |
| 二连击 | 1 | 4×2 伤害 |
| 盾击 | 2 | 护盾值等同伤害 |

## 操作

点击卡牌 → 选择目标（如需要）→ 结束回合

## 特性

- 10 种可用卡牌
- 5 种敌人（含 Boss）
- 8 层递进难度
- 回合制战斗 + 自建牌组
- 战后奖励选择
- 毒伤 / 力量增益 / 溅射

## 已知限制

- 无音效
- 无地图路径选择
- 无存档
