# 平台跳跃

横版卷轴平台跳跃游戏，跑跳冲刺到达终点旗。

## 玩法

- 控制角色在关卡中移动和跳跃
- 收集金币增加分数
- 避开尖刺（红色三角）
- 到达终点旗（红旗）即过关
- 掉入深渊或碰到尖刺失去一条命
- 3 条命用完游戏结束

## 操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动 | ← → / A D | 左右按钮 |
| 跳跃 | 空格 / ↑ / W | 跳按钮 |
| 重玩 | 重玩按钮 | 重玩按钮 |

## 特性

- 3 个手工设计关卡
- 卷轴镜头跟随
- 金币收集系统
- 尖刺陷阱
- 物理跳跃手感
- 键盘 + 触控双模式

## 已知限制

- 无音效
- 无敌人 AI
- 关卡数量有限
