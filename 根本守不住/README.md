### 游戏模式
- **对抗模式**：无配置脚本，敌方 AI 自由出兵，可无限对玩。
- **战役模式**：关卡选择 → 按照脚本波次攻防 → 结算奖励。
  - 新手营地：敌方仅用剑士和弓箭手
  - 森林小路：敌方引入盾兵
  - 沼泽前线：敌方引入法师

### 资源与平衡
- 能量：0-10，基础回复 1/s，90秒后 1.5/s
- 基地血量：2000-2200（根据关卡）
- 胜利：进度条 +100/-100 或基地血量归零

### 核心机制
- 双方基地分居屏幕左右
- 消耗能量召唤单位，单位自动前进并交战
- 顶部进度条实时拉扯（红蓝双向）
- 单位击杀、基地技能都推动进度条
